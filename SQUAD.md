# Decisao & Pensamento Profundo

> Squad de 4 mentes para pensar melhor: Daniel Kahneman (vieses cognitivos), Kapil Gupta (verdade radical), Ray Kurzweil (exponencial), Jesus Cristo (lideranca servidora). Routing inteligente envia seu dilema para o pensador certo.

## Como Funciona

O squad analisa seu dilema e roteia automaticamente para o pensador mais qualificado:

```
/deep-thinking-minds Recebi proposta para empreender. Nao sei se aceito.
→ [Routing: Daniel Kahneman | Confianca: high]
→ Identifica vieses: status quo bias, loss aversion, optimism bias
→ Handoff para Kapil Gupta: "Voce quer empreender ou fugir do emprego?"
```

## Agent Roster

| Agente | Dominio | Quando Ativa |
|--------|---------|-------------|
| Daniel Kahneman | Vieses cognitivos, System 1/2, ciencia da decisao | Decisoes dificeis, incerteza, risco, heuristicas |
| Kapil Gupta | Verdade, anti-prescricao, clareza radical | Busca por autenticidade, desapego de formulas |
| Ray Kurzweil | Pensamento exponencial, futurismo, singularidade | Visao de futuro, tendencias, convergencia tecnologica |
| Jesus Cristo | Lideranca servidora, parabolas, graca+verdade | Proposito, etica, conflitos, influencia transformadora |

## Workflows Integrados

1. **Decisao de Alto Impacto** — Vieses (Kahneman) + verdade (Gupta) + consequencias (Kurzweil) + proposito (Jesus)
2. **Clareza Existencial** — Destruir prescricoes (Gupta) + reconectar proposito (Jesus) + stress-test (Kahneman)
3. **Visao Estrategica** — Tendencias (Kurzweil) + reality check (Kahneman) + verdade pessoal (Gupta)
